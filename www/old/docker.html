Docker y sus Contenedores

A continuación se muestra una lista de los contenedores presentes en el archivo Docker Compose y una breve descripción de cada uno:

Contenedor Nginx

Este contenedor es responsable de servir el contenido web de la aplicación. Utiliza el servidor web Nginx para gestionar las solicitudes HTTP y las HTTPS.

Contenedor LetsEncrypt

Este contenedor se encarga de generar y renovar automáticamente los certificados SSL/TLS para habilitar conexiones seguras a través de HTTPS.

Contenedor MySQL

Este contenedor aloja la base de datos MySQL utilizada por la aplicación. Proporciona un almacenamiento persistente para los datos de la aplicación.

Contenedor PHP-FPM

Este contenedor ejecuta el intérprete PHP y maneja las solicitudes PHP enviadas por el servidor web Nginx.
